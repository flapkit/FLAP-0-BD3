# Manufacturing Release Files - BD3 Rev 01

This directory contains manufacturing-ready files for BD3 Rev 01.

## Directory Structure

- **gerbers/**: Gerber files for PCB fabrication
- **drill/**: Drill files for PCB manufacturing
- **pick-and-place/**: Pick and place files for automated assembly
- **position-files/**: Component position files
- **bom/**: Bill of materials for manufacturing
- **assembly-drawings/**: Assembly drawings and documentation
- **test-notes/**: Manufacturing test notes and procedures

## Usage

These files are production-ready and can be sent directly to PCB manufacturers and assembly houses.

## Version

- **Hardware Version**: BD3 Rev 01
- **Release Date**: See CHANGELOG.md in repository root
