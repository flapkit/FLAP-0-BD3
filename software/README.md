# Firmware for FLAP-0 BD3

Firmware, HDL designs, and related tooling for FLAP-0 BD3 are maintained in a separate repository.

## Firmware Repository

**Repository**: [FLAP-0-BD3-Firmware](https://github.com/flapkit/FLAP-0-BD3-Firmware)

## Contents

The firmware repository contains:

- **Reference HDL designs**: Production-ready HDL reference implementations
- **Example firmware**: Example designs and tutorials
- **Tooling scripts**: Build automation, synthesis scripts, and utilities
- **Build instructions**: Documentation for building and programming the FPGA
- **CI/CD**: Continuous integration for synthesis and bitstream generation

## Getting Started

Please visit the [FLAP-0-BD3-Firmware repository](https://github.com/flapkit/FLAP-0-BD3-Firmware) for:
- Installation instructions
- Build documentation
- Example designs
- API documentation

## Hardware Repository

This directory is part of the hardware repository. For hardware design files, schematics, PCB layouts, and manufacturing files, see the main [FLAP-0-BD3 repository](https://github.com/flapkit/FLAP-0-BD3).
