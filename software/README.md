# Software for FLAP-0 BD3

Software, HDL designs, and related tooling for FLAP-0 BD3 are maintained in a separate repository.

## Software Repository

**Repository**: [FLAP-0-BD3-Software](https://github.com/flapkit/FLAP-0-BD3-Software)

## Contents

The software repository contains:

- **Reference HDL designs**: Production-ready HDL reference implementations
- **Example software**: Example designs and tutorials
- **Tooling scripts**: Build automation, synthesis scripts, and utilities
- **Build instructions**: Documentation for building and programming the FPGA
- **CI/CD**: Continuous integration for synthesis and bitstream generation

## Getting Started

Please visit the [FLAP-0-BD3-Software repository](https://github.com/flapkit/FLAP-0-BD3-Software) for:
- Installation instructions
- Build documentation
- Example designs
- API documentation

## Hardware Repository

This directory is part of the hardware repository. For hardware design files, schematics, PCB layouts, and manufacturing files, see the main [FLAP-0-BD3 repository](https://github.com/flapkit/FLAP-0-BD3).
